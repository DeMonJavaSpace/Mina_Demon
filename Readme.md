### MinaSocket

基于MinaSocket的服务端+PC客户端+Android端，多端通讯实现。

1. 服务端&PC客户端业务代码使用Java实现，界面使用JavaFx实现。
2. Android端使用Kotlin。
3. Socket通信基于Mina Socket实现```mina-core-2.0.7.jar```。
